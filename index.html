How to use

The source data for this visualization comes from a Google Docs Spreadsheet. To create your own timeline use this Google Docs template.

Google Doc Template

Once you have your information properly organized in columns [order] [characther] [group] you must get a RSS link. To do that, go to your Spreadsheet and select:

  1. File...
  2. Publish to the web
  3. Select the sheet
  4. Click on "Start Publishing" (if not selected)
  5. Click on [x] automatically republish
  6. Get a link to the published data: RSS
  7. Once you have the link, past it in the URL field (replace the one already pasted by default)
